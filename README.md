# Food Ordering System

A C++ console-based food ordering system built for a university programming assignment. 

# Technologies Used
- C++

# Features
- You will be presented with a table of available food on your order page.
- Once you have found the food you want to order, just input the code of the food, and
you can see the food is in your trolley.
- If you want to add on other foods, just input ‘Y’ or ‘y’ to continue ordering. If not, the
system will turn to the payment page.
- Review your order total. You will be required to make payment at the counter. After
making payment, you will receive a confirmation that your order has been
successfully placed.
- After placing your order, you can see how much time it takes to prepare. You will
receive updates on the preparation and estimated completion time of your order.Display menu with food items and prices
